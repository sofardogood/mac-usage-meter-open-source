# Mac Usage Meter

macOS のメニューバーから、Mac 全体の消費電力とネットワーク使用量を確認できるローカル常駐アプリです。

## 主な機能

- `powermetrics` による Mac 全体の消費電力（W / kWh）の記録
- Wi-Fi の送受信量、日次・月次の集計、概算料金
- アプリ名と接続先ホスト別の通信量の可視化
- 利用先別通信量の円グラフ・棒グラフ
- 通信活動量を基にした、利用先ごとの推定電力表示
- CSV エクスポート

接続先は macOS の接続統計から取得します。HTTPS の通信内容を復号・保存せず、接続先ホスト名のみを表示します。そのため CDN のホスト名が表示される場合があります。利用先別の電力はアプリ単体を直接計測した値ではなく、Mac 全体の実測電力を通信活動量で配分した推定値です。

## 動作環境

- macOS 13 以降
- Swift 5.9 以降

消費電力の取得には、ローカル Helper と管理者権限が必要です。通信量の収集・表示は Helper なしでも動作します。
